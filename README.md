# InternSavy Java Tasks 🚀

Welcome to the InternSavy Java Tasks repository! This project contains a collection of Java programming assignments completed as part of the InternSavy internship program.

## 📂 Project Structure

The repository is organized into distinct tasks:

- **Task 1: Temperature Converter** - A CLI-based tool to convert temperatures between Celsius and Fahrenheit.
- **Task 2: Tic-Tac-Toe** - A Tic-Tac-Toe game implementation (Currently a work-in-progress/placeholder).
- **Task 3: GUI Calculator** - A fully functional Graphical User Interface (GUI) calculator built using Java Swing.

## 🛠️ Prerequisites

Before executing the programs, ensure you have the following installed on your system:

- **Java Development Kit (JDK): Version 8 or higher.**

Validate your Java installation by running:

```bash
java -version
javac -version
```

## ⚙️ How to Compile and Run

Because the Java files are structured within a specific package (`InternSavyTasks.InternSavy.TaskX`), all commands must be executed from the root directory of the project.

First, open your Command Prompt (or Terminal) and navigate to the project's root folder:

```bash
cd c:\Dishi\Repod
```

## ▶️ Task 1: Temperature Converter

This is a console application that seamlessly handles temperature conversions based on user input.

### 1. Compile the code:

```bash
javac InternSavyTasks\InternSavy\Task1\TemperatureConverter.java
```

### 2. Run the application:

```bash
java InternSavyTasks.InternSavy.Task1.TemperatureConverter
```

### 3. Usage:

Once it runs, input 1 or 2 in the console to choose your conversion type, then enter the temperature value.

## ▶️ Task 3: GUI Calculator

This application features a complete graphical interface (using javax.swing) for basic arithmetic operations (+, -, *, /) along with a clear and delete button.

### 1. Compile the code:

```bash
javac InternSavyTasks\InternSavy\Task3\Calculator.java
```

### 2. Run the application:

```bash
java InternSavyTasks.InternSavy.Task3.Calculator
```

### 3. Usage:

A calculator window will pop up automatically. Click on the buttons to perform mathematical operations just like a standard desktop calculator.

## 📝 Notes

- Task 2 (TicTacToe.java) is currently an empty file and serves as a placeholder for future implementation.
- If you encounter a ClassNotFoundException, make sure you are executing the java commands precisely from the `c:\Dishi\Repod` directory, as package resolution depends on your current working directory.
